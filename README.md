# 🎛️ Fonk Creator - Free Beat Maker

Make phonk, trap, boom bap beats in your browser. Export MP3. No signup.

**Live Demo:** https://yourusername.github.io/fonk-creator/

## ✨ Features

- **8-Track Sequencer** - Kick, Snare, Hi-Hat, 808, Clap, Crash, Percs, FX
- **Piano Roll** - Add bass/melody notes
- **4 Drum Kits** - Trap, Boom Bap, Phonk, 808 Heavy
- **Custom Samples** - Upload your own WAV/MP3
- **Mixer** - Volume per track
- **Export MP3** - Download your beat instantly
- **Save/Load Projects** - JSON format
- **BPM Control** - 60-200 BPM
- **100% Client-Side** - No backend, works on GitHub Pages

## 🚀 Deploy

1. Fork this repo
2. Settings → Pages → Source: main → Save
3. Visit `https://yourusername.github.io/fonk-creator/`

## 🎵 How To Use

1. **Click pads** to play sounds live
2. **Click sequencer steps** to program beat
3. **Press Play** to hear loop
4. **Adjust BPM** for tempo
5. **Use Piano Roll** for basslines
6. **Export MP3** when done

## 🎹 Keyboard Shortcuts

- `Space` - Play/Pause
- `1-8` - Trigger drum pads
- `C` - Clear sequencer
- `R` - Randomize beat

## 📦 Sample Packs

Default kit uses Tone.js synths. To add real samples:

1. Upload WAV/MP3 via "Add Your Sound"
2. Samples save to browser only
3. For production, host samples on GitHub and update URLs in `app.js`

## 🔧 Tech Stack

- **Tone.js** - Web Audio API wrapper
- **LameJS** - MP3 encoding in browser
- **Vanilla JS** - No frameworks
- **GitHub Pages** - Free hosting

## 🎧 Tips

- Start with 140 BPM for phonk/trap
- Layer kick + 808 for heavy bass
- Use hi-hat rolls on steps 3,7,11,15
- Export at 128kbps MP3 for web, 320kbps for upload
